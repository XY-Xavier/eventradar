Multi-city networking events dashboard for tech, business, and startup professionals.

## Features

- **4 Major Cities**: San Francisco/Bay Area, Los Angeles, New York City, Chicago
- **Smart Filters**: Search, price range, time window
- **Clean UI**: Modern, responsive design built with Tailwind CSS
- **Demo Data**: Pre-loaded with curated networking events

## Live Demo

Visit: [xy-xavier.github.io/eventradar](https://xy-xavier.github.io/eventradar)

## Local Development

```bash
# Clone the repo
git clone https://github.com/XY-Xavier/eventradar.git
cd eventradar

# Serve locally (Python 3)
python3 -m http.server 8000

# Or use any static server
npx serve .
```

Then open: http://localhost:8000

## Tech Stack

- Pure HTML/CSS/JavaScript (no frameworks)
- Tailwind CSS (CDN)
- Font Awesome icons
- Static JSON data

## Data Sources

Currently using demo data. Future versions will integrate:
- Eventbrite API
- Meetup API  
- Luma API
